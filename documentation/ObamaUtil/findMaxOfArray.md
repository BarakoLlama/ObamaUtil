# findMaxOfArray

### array
Array(Number()): The array to find the largest number in
## Example
```javascript
const ObamaUtil = require('ObamaUtil')
const obama = new ObamaUtil.ObamaUtil()
obama.findMinOfArray([-2, 6, 0, 19])
```
