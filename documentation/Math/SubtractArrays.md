# SubtractArrays
## Parameters
### n1
Array(Number()): The first matrix perform operations on
### n2
Array(Number()): The second matrix to perform operations on
## Example
```javascript
const ObamaUtil = require('ObamaUtil')
var array1 = [1, 3, 3]
var array2 = [2, 2, 4]
var myArray = ObamaUtil.Math.SubtractArrays(array1, array2)
// Returns [-1, 1, -1]
```
